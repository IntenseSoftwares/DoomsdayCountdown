# DoomsdayCountdown

A minimal Marvel Avengers **Doomsday countdown** website made to be used as a **Web Wallpaper** in **Lively Wallpaper**.

Live site: https://intensesoftwares.github.io/DoomsdayCountdown/

---

## What this is
- Fullscreen **video background**
- **Countdown timer** (Months / Days / Hours / Minutes / Seconds)
- No extra UI elements — clean and wallpaper-ready

---

## Use in Lively Wallpaper (Step-by-step)

### Method 1 (Recommended): Add as a Web Wallpaper
1. Open **Lively Wallpaper**
2. Click **+ Add Wallpaper** (top-left)
3. Choose **Web / Webpage**
4. In the URL box, paste:
(`https://intensesoftwares.github.io/DoomsdayCountdown/`)
5. Click **OK / Add**
6. Select the newly added wallpaper to apply it

✅ Done — your countdown webpage will run as your live wallpaper.

---

### Method 2: Better performance (Local files)
If you want smoother playback and less network dependency, you can use it locally.

1. Open this repo on GitHub
2. Click **Code** → **Download ZIP**
3. Extract the ZIP anywhere (example: `D:\Wallpapers\DoomsdayCountdown\`)
4. Open **Lively Wallpaper**
5. Click **+ Add Wallpaper**
6. Choose **HTML / Local Webpage**
7. Select the extracted `index.html`

✅ This runs almost offline and performs better.

---

## Tips (Lively)
- A "Ticking" audio will play when theere is no application occupying the desktop view.
- If you want less CPU usage, lower the update rate in `script.js` (example: update every `1000ms`).
