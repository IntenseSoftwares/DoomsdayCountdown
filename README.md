# DoomsdayCountdown

A minimal Marvel Avengers–style “Doomsday” countdown webpage designed to be used as a **Lively Wallpaper** web link.

It shows:
- A **full-screen looping video background** (`DoomsdayBG.mp4`)
- A **clean countdown timer** (Months : Days : Hours : Min : Sec)
- No extra UI elements, buttons, or sections
